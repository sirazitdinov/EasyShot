# EasyShot — Chrome Extension

EasyShot — расширение для Chrome, которое предоставляет расширенные возможности для создания скриншотов и редактирования изображений. Позволяет захватывать полные экраны, выбирать области для скриншотов и редактировать захваченные изображения с помощью инструментов аннотации.

## Особенности

- **Полноэкранные скриншоты**: быстрый захват всего текущего экрана
- **Выбор области**: интуитивный выбор произвольной области для скриншота
- **Встроенный редактор изображений**: комплексный редактор на Canvas API
- **Пиксельная линейка**: инструмент для измерения элементов на веб-страницах
- **Поддержка горячих клавиш**: быстрый доступ к функциям (`Ctrl+Shift+R` для переключения линейки)
- **Инструменты редактирования**:
  - Обрезка (crop)
  - Размытие (blur)
  - Прямоугольник (rectangle)
  - Маркер (highlighter)
  - Линия со стрелкой
  - Текст
- **Управление слоями**: работа с несколькими слоями аннотаций
- **История действий**: undo/redo с оптимизированным хранением состояний

## Установка и запуск

### Требования

- Браузер Chrome
- Включённый режим разработчика в разделе «Расширения»

### Установка

1. Клонируйте или скачайте файлы расширения
2. Откройте Chrome и перейдите на страницу `chrome://extensions`
3. Включите «Режим разработчика» в правом верхнем углу
4. Нажмите «Загрузить распакованное» и выберите директорию расширения
5. Расширение появится в панели инструментов

### Разработка

1. Внесите изменения в исходные файлы
2. На странице `chrome://extensions` нажмите значок обновления для расширения
3. Протестируйте функциональность в новой вкладке

## Архитектура

### Фоновый скрипт (`background.js`)

- Управляет жизненным циклом расширения и обменом сообщениями
- Обрабатывает операции захвата скриншотов
- Координирует взаимодействие между компонентами
- Реализует переключение пиксельной линейки

### Контент-скрипты (`content-scripts/`)

- **content.js**: внедряет пиксельную линейку (`PixelRuler`) в веб-страницы
- **overlay.js**: создаёт оверлей для выбора области скриншота

### Всплывающее окно (`popup/`)

- Простой UI для запуска операций скриншота
- Кнопки для полного экрана, выбора области, редактора и переключения линейки

### Редактор изображений (`editor/`)

Редактор построен по модульной архитектуре с чётким разделением ответственности:

- **`EditorCore`** — координация, инициализация, glue-код (~1000 строк)
- **`CanvasManager`** — рендеринг холста, dirty region, crop overlay
- **`SelectionRenderer`** — отрисовка ручек выделения
- **`EventManager`** — обработчики мыши/клавиатуры, drag/resize через `requestAnimationFrame`
- **`LayerManager`** — CRUD слоёв, порядок, активный слой
- **`HistoryManager`** — undo/redo с оптимизированным хранением (`OffscreenCanvas`, динамический лимит по памяти)
- **`*Tool`** — каждый инструмент реализует `renderLayer`, `hitTest`, `getBounds`
- **`renderers/*`** — низкоуровневый canvas-код, используемый инструментами
- **`crop/*`** — логика применения кропа и пересчёта координат слоёв

## Использование

### Создание скриншота

1. Нажмите на значок расширения в панели инструментов Chrome
2. Выберите один из вариантов:
   - «Скриншот всего экрана» — создаёт скриншот всей видимой области
   - «Скриншот области» — позволяет выбрать область для захвата
   - «Открыть редактор» — открывает встроенный редактор изображений
   - «Линейка (Ctrl+Shift+R)» — переключает пиксельную линейку

### Пиксельная линейка

1. Активируйте линейку с помощью кнопки или горячих клавиш (`Ctrl+Shift+R`)
2. Нажмите и перетащите мышь, чтобы создать прямоугольник измерения
3. Используйте клавишу пробел для перемещения выделенного прямоугольника
4. Нажмите `Escape` для отключения линейки

### Редактирование изображений

1. Откройте редактор изображений
2. Загрузите изображение или вставьте из буфера обмена (`Ctrl+V`)
3. Используйте инструменты для аннотации:
   - **Обрезка**: выберите область и примените кроп
   - **Размытие**: настройте радиус и выделите область
   - **Рамка**: прямоугольная рамка с настраиваемым цветом
   - **Маркер**: заливка с прозрачностью
   - **Стрелка**: линия со стрелкой
   - **Текст**: текстовые аннотации с настройками размера и цвета
4. Управляйте слоями через панель слоёв
5. Используйте `Ctrl+Z` / `Ctrl+Shift+Z` для отмены и повтора
6. Нажмите `Delete` для удаления активного слоя
7. Нажмите `Escape` для сброса выделения
8. Сохраните результат с помощью кнопки сохранения

## API и разрешения

Расширение использует следующие разрешения Chrome:

- `activeTab` — доступ к активной вкладке
- `scripting` — внедрение скриптов
- `downloads` — сохранение файлов
- `tabs` — взаимодействие с вкладками

# EasyShot - Chrome Extension

EasyShot - это мощное и удобное расширение для браузера Chrome, которое предоставляет расширенные возможности для создания скриншотов с встроенным редактором изображений. Расширение позволяет захватывать полные экраны, выбирать области для скриншотов и редактировать захваченные изображения с различными инструментами аннотации.

## Особенности

- **Полноэкранные скриншоты**: Быстрый захват всего текущего экрана
- **Выбор области**: Интуитивный выбор произвольной области для скриншота
- **Встроенный редактор изображений**: Комплексный редактор с различными инструментами аннотации
- **Пиксельная линейка**: Инструмент для измерения элементов на веб-страницах
- **Поддержка горячих клавиш**: Быстрый доступ к функциям (Ctrl+Shift+R для переключения линейки)
- **Инструменты редактирования**:
  - Обрезка
  - Размытие
  - Выделение
  - Рисование стрелок
  - Добавление текста
- **Управление слоями**: Возможность работы с несколькими слоями изображений
- **История действий**: Поддержка отмены/повтора действий

## Архитектура

Расширение следует типичной архитектуре расширений Chrome с несколькими компонентами:

### Файл манифеста (`manifest.json`)
- Определяет основные метаданные расширения
- Объявляет разрешения и API, необходимые для работы
- Конфигурирует фоновый сервис-воркер и контент-скрипты

### Фоновый скрипт (`background.js`)
- Управляет жизненным циклом расширения и обменом сообщениями
- Обрабатывает операции захвата скриншотов
- Координирует взаимодействие между различными компонентами расширения
- Реализует функцию переключения пиксельной линейки

### Контент-скрипты (`content-scripts/`)
- **content.js**: Внедряет функциональность пиксельной линейки в веб-страницы
- **overlay.js**: Создает оверлей для выбора области скриншота

### Интерфейс всплывающего окна (`popup/`)
- Простой пользовательский интерфейс для запуска операций скриншота
- Кнопки для полного экрана, выбора области, редактора и переключения линейки

### Редактор изображений (`editor/`)
- Комплексный интерфейс редактирования изображений
- Множество инструментов аннотации (обрезка, размытие, выделение, линии, текст)
- Система управления слоями
- Функциональность истории/отмены
- Возможности экспорта с выбором формата

### Интерфейс захвата (`capture/`)
- Специализированный интерфейс для выбора области
- Визуальный оверлей для выбора регионов скриншота

## Технологии

- JavaScript (ES6+)
- HTML/CSS
- Canvas API для манипуляций с изображениями
- Chrome Extension APIs (runtime, tabs, downloads, scripting, windows)

## Установка и запуск

### Необходимые условия
- Браузер Chrome
- Включенный режим разработчика в разделе "Расширения"

### Установка
1. Клонируйте или скачайте файлы расширения
2. Откройте Chrome и перейдите на страницу `chrome://extensions`
3. Включите "Режим разработчика" в правом верхнем углу
4. Нажмите "Загрузить распакованное" и выберите директорию расширения
5. Расширение должно появиться в вашей панели инструментов

### Разработка
Для разработки и тестирования расширения:
1. Внесите изменения в исходные файлы
2. На странице `chrome://extensions` нажмите значок обновления для расширения
3. Протестируйте функциональность в новой вкладке

### Тестирование
Проект использует Vitest для запуска тестов:

```bash
npm test          # Запуск тестов
npm run test:ui   # Запуск тестов с UI
npm run coverage  # Запуск тестов с покрытием кода
```

    - `TextTool.js` - Добавление текста

- `popup/` - Всплывающее окно расширения
- `tests/` - Тесты (Vitest)

## Использование

### Создание скриншота
1. Нажмите на значок расширения в панели инструментов Chrome
2. Выберите один из вариантов:
   - "Скриншот всего экрана" - создает скриншот всей видимой области
   - "Скриншот области" - позволяет выбрать область для захвата
   - "Открыть редактор" - открывает встроенный редактор изображений
   - "Линейка (Ctrl+Shift+R)" - переключает пиксельную линейку

### Использование пиксельной линейки
1. Активируйте линейку с помощью кнопки или горячих клавиш (Ctrl+Shift+R)
2. Нажмите и перетащите мышь, чтобы создать прямоугольник измерения
3. Используйте клавишу пробел для перемещения выделенного прямоугольника
4. Нажмите Escape для отключения линейки

### Редактирование изображений
1. Откройте редактор изображений
2. Загрузите изображение или откройте его из истории
3. Используйте различные инструменты для аннотации:
   - Кадрирование: Выберите инструмент кадрирования и выделите область
   - Размытие: Выберите инструмент размытия и настройте радиус
   - Выделение: Выделите области с настраиваемым цветом
   - Стрелка: Добавьте стрелки для указания
   - Текст: Добавьте текстовые аннотации с настройками стиля
4. Сохраните результат с помощью кнопки сохранения

## Разработка

### Соглашения по коду
- Файлы JavaScript используют camelCase
- CSS-классы используют kebab-case
- Константы записываются в UPPER_SNAKE_CASE
- Переменные и функции используют camelCase

### Передача сообщений
- Используйте `chrome.runtime.sendMessage` для связи между фоновым скриптом и всплывающим окном
- Используйте `chrome.tabs.sendMessage` для связи между фоновым скриптом и контент-скриптами
- Всегда обрабатывайте потенциальные ошибки в ответах на сообщения

### Архитектура редактора
Редактор использует подход с уровнями:
- Базовый уровень: Оригинальное изображение
- Уровни аннотаций: Обрезка, размытие, выделение, линии, текст
- Оверлей выделения: Интерактивный интерфейс для создания/редактирования аннотаций
- Историческая система: Функциональность отмены/повтора

### Система инструментов
Каждый инструмент редактора следует согласованному шаблону:
- Отдельная реализация класса
- Конфигурация параметров
- Методы рендеринга на холсте
- Манипуляции с маркерами для изменения размера/перемещения

### Пиксельная линейка
Функция пиксельной линейки включает:
- Визуальный оверлей с перетаскиваемым выделением
- Отображение размеров выбранных областей в реальном времени
- Горячие клавиши (пробел для перемещения, Escape для закрытия)
- Маркеры изменения размера для точной настройки

## API и разрешения

Расширение использует следующие разрешения Chrome:
- `activeTab` - для доступа к активной вкладке
- `scripting` - для внедрения скриптов
- `downloads` - для сохранения файлов
- `tabs` - для взаимодействия с вкладками
